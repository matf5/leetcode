344.Reverse String 
Write a function that takes a string as input and returns the string reversed.

Example:
Given s = "hello", return "olleh".

solution:

public class Solution {
    public String reverseString(String s) {
        int len  = s.length();
        String ans = "";
        for(int i=0;i<len;i++){
            ans += s.charAt(i);
        }
        return ans;
    }
}

string.charAt(i) 返回字符串在该位置的字符
